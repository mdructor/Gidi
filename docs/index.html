Gidi

a new way to play

By converting controller input, like button presses and stick movements, into MIDI messages, we can connect to a wide array of MIDI processors in any DAW and transform a controller into an instrument.
Gidi is a tool to easily manage and control this interface.

Gidi's development started in April 2019 and is still an active and ongoing process. If you are interested in learning more about Gidi, please reach out to me at masondructor@gmail.com.
